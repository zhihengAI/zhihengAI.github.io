ubuntu 离线安装 ffmpeg 7.0

先下载ffmpeg7压缩包

1

wget https://ffmpeg.org/releases/ffmpeg-7.0.tar.gz

接着安装相应依赖

1

sudo apt-get install yasm

然后完成安装

1
2
3
4
5
6
7

tar -zxvf ffmpeg-7.0.tar.gz

# 安装
cd ffmpeg-7.0
./configure --enable-shared --prefix=/usr/local/ffmpeg

sudo make && sudo make install

下面是安装过程中遇到的错误

1. ffmpeg查看版本报错error while loading shared libraries: libavdevice.so.59 61

解决方法：

  1. vim /etc/ld.so.conf
  2. 在最后一行加入/usr/local/ffmpeg/lib，具体看自己的安装目录
  3. 刷新配置ldconfig即可
